# Validation
Реализовал и сравнил различные стратегии валидации ML-моделей (random split, K-Fold, TimeSeriesSplit), исследовал методы отбора признаков и подбора гиперпараметров (Grid Search, Optuna). Построил и интерпретировал линейные модели регрессии (Linear, Lasso, ElasticNet), обеспечив корректную работу с временными данными и исключив утечки.
